📥 Import Soal via CSV
📋 Panduan Pengisian Template
  1. Klik Unduh Template CSV Soal di bawah ini.
  2. Isi data menggunakan Excel atau aplikasi spreadsheet lainnya.
  3. Kolom tipe_soal wajib diisi: pilihan_ganda, benar_salah, matching, audio, teks_arab, atau isian_singkat.
  4. Kolom pilihan: pisahkan dengan |, akhiri pilihan benar dengan tanda * (contoh: A*|B|C|D).
  5. Kolom levels: pisahkan level halaqah dengan koma (contoh: Level 1,Level 2).
  6. Simpan file dengan format .csv (UTF-8, pemisah titik koma / semicolon).
