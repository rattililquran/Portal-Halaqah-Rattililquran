📥 Import Data via CSV
📋 Cara Pakai
  1. Klik Unduh Template CSV di bawah.
  2. Isi data di Excel atau aplikasi spreadsheet — guru dulu, lalu murid.
  3. Kolom nama_halaqah wajib diisi untuk murid agar langsung terdaftar di kelas.
  4. Save As → format .csv (UTF-8).
  5. Upload file, periksa preview, klik Import Data.
