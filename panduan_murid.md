## 1. Cara Login dengan NIS

**NIS** adalah Nomor Induk Santri — kode unik yang diberikan admin saat mendaftar. Contoh: `RTL2418000001`.

### Langkah Login
1. Buka portal melalui tautan dari guru atau admin
2. Di halaman pertama, ketik **NIS** di kolom pertama
3. Ketik **Password** di kolom kedua
   - Jika belum pernah set password, kosongkan saja dan langsung tekan **Masuk**
4. Tekan tombol **Masuk**
5. Jika berhasil, akan masuk ke halaman **Beranda**

> ⚠️ Jika muncul pesan merah "NIS atau password salah", periksa kembali NIS. Hubungi guru atau admin jika tidak bisa masuk.

## 2. Memahami Beranda / Dashboard

### Kartu "Perjuanganku Bersama Al-Qur'an"

Kartu besar berwarna biru di atas menampilkan progress level:
- Level sekarang → Level tujuan
- Bar kemajuan dari 40 sesi KBM Reguler
- Persentase ketercapaian dan sisa sesi

> 💡 Ketuk kartu ini untuk melihat rincian lengkap kehadiran per status.

### 4 Kartu Statistik
| Ikon | Keterangan |
|------|-----------|
| % Hadir | Persentase kehadiran dari semua sesi KBM |
| Sesi | Total sesi KBM yang tercatat |
| Poin Adab | Poin berdasarkan penilaian adab |
| Poin Kamera | Poin berdasarkan status kamera |

### Kartu At-Tibyan
Kartu emas-oranye di bawah progress bar. Ketuk untuk langsung membuka halaman At-Tibyan dan melihat materi kajian mingguan serta kehadiran kamu.

### Grafik Kehadiran 6 Bulan
Grafik batang yang menampilkan kehadiran per bulan:
- **Kotak hijau besar** di atas = rata-rata kehadiran 6 bulan
- **Batang hijau** = kehadiran ≥ 80% (sangat baik)
- **Batang kuning/amber** = kehadiran 50-79% (cukup)
- **Batang merah** = kehadiran < 50% (perlu ditingkatkan)
- **Batang abu-abu** = bulan tanpa sesi (libur)
- **Bagian ungu** di atas batang = porsi terlambat
- Garis putus hijau = target 80%, garis amber = batas 50%
- Bulan ini ditandai titik biru (●)
- Ketuk batang → tampil detail: hadir, terlambat, alpa, total sesi

### Kartu Status SPP
- Hijau = sudah lunas
- Merah = belum lunas

### Latihan Mandiri Aktif
Kartu PR terbaru dari guru. Ketuk **Lihat** untuk melihat semua PR.

### Pengumuman
Pengumuman terbaru dari guru atau admin.

### Akses Cepat
- **KhatamKu** — tracker khatam Al-Qur'an
- **Portal SPP** — konfirmasi pembayaran SPP

## 3. Tab Riwayat KBM

Tab **Riwayat** (ikon jam) menampilkan catatan seluruh sesi KBM.

### Kalender Kehadiran
Warna tanggal sesuai kehadiran:
- Hijau = Hadir
- Merah = Alpa
- Kuning = Izin atau Terlambat

Ketuk tanda **‹** atau **›** untuk pindah bulan. Ketuk tanggal berwarna untuk lihat detail sesi hari itu.

### Daftar Riwayat Sesi
Setiap sesi menampilkan tanggal, nomor sesi, materi, dan badge status. Ketuk untuk membuka detail:
- Status hadir dan nilai adab/kamera
- Materi yang dipelajari
- Koreksi tahsin dari guru
- Catatan khusus dari guru
- PR/Latihan mandiri beserta deadline

### Muat Lebih Banyak
Default 10 riwayat terakhir. Ketuk tombol untuk muat lebih banyak.

## 6. Menu At-Tibyan

At-Tibyan adalah kajian khusus mingguan untuk seluruh murid Rattililqur'an. Buka dari kartu emas di Beranda atau Sidebar Menu → At-Tibyan.

### Tab Kajian

**Kartu Kajian Terbaru** (banner merah di atas):
- Pertemuan terakhir selalu di-pin di paling atas
- Langsung terbuka saat halaman dibuka

**Filter Pertemuan:**
Tombol "Semua / Pertemuan 1 / Pertemuan 2 / ..." untuk filter materi tertentu.

**Isi setiap kartu:**
- Tanggal pertemuan
- Pemateri / Ustadz
- Materi Pembahasan
- Nasihat Aplikatif
- Kolom lain jika ada

**Kotak Pencarian:**
Cari keyword dari materi atau nasihat di semua pertemuan.

**Tombol Salin dan Bagikan:**
| Tombol | Fungsi |
|--------|--------|
| Salin | Copy teks materi ke clipboard (bisa tempel di WA, Notes, dll) |
| Bagikan | Buka menu berbagi bawaan HP (WA, IG, Telegram, dll) |

### Tab Kehadiranku

Menampilkan riwayat kehadiran kamu di kajian At-Tibyan:
- Kotak besar: % hadir dari total sesi + keterangan
  - Hijau = ≥ 80% (kehadiran baik)
  - Amber = 60-79% (cukup)
  - Merah = < 60% (perlu ditingkatkan)
- Daftar per pertemuan dengan badge: Hadir / Terlambat / Izin / Alpa

## 7. Tab Raport

Raport hanya bisa diakses melalui **Sidebar Menu** (lihat bagian 8).

### Melihat Raport
- Nilai akhir dan predikat (Mumtaz, Jayyid Jiddan, Jayyid, Maqbul)
- Rincian nilai per komponen dan bobotnya
- Termasuk komponen "Kehadiran At-Tibyan" jika dikonfigurasi

### Unduh PDF
1. Ketuk tombol **Preview** pada raport
2. Cek pratinjau dalam format A4
3. Ketuk **Unduh PDF** untuk simpan ke HP

> ⚠️ Raport hanya muncul setelah dipublikasikan oleh admin.

## 8. Lonceng Notifikasi dan Sidebar Menu

### Lonceng Notifikasi
Ikon lonceng di pojok kanan atas. Angka merah kecil = ada notifikasi baru.

**Jenis notifikasi:**
- Peringatan keaktifan (kehadiran berkurang)
- Peringatan kritis (kehadiran sangat rendah)

Ketuk lonceng untuk melihat semua catatan. Catatan tersimpan permanen sebagai rekam jejak.

### Sidebar Menu
Ketuk ikon hamburger (pojok kiri atas) atau tombol logo di tengah menu bawah.

| Menu | Fungsi |
|------|--------|
| Profil Saya | Lihat dan ubah data profil, ganti foto |
| Notifikasi | Semua catatan keaktifan |
| At-Tibyan | Kajian mingguan dan kehadiran |
| Raport | Nilai akhir yang dipublikasikan |
| Pengumuman | Info dari guru dan admin |
| Assessment Mandiri | Hanya muncul untuk Level 1 & 2 |
| Install Aplikasi | Panduan install PWA di HP |
| Ganti Password | Ubah password akun |
| Tentang Rattililqur'an | Info lembaga, donasi, sosmed |
| Keluar | Logout dari aplikasi |

## 9. Ganti Foto Profil

1. Buka **Profil Saya** dari sidebar
2. Ketuk lingkaran foto/avatar (ada ikon kamera kecil di pojok)
3. Pilih foto dari galeri HP
4. Foto dikompres otomatis sebelum dikirim
5. Ketuk **Simpan**

> 💡 Foto akan muncul di menu navigasi bawah sebagai avatar pribadi.

### Edit Data Profil Lainnya
Di halaman Profil, bisa update:
- No HP, Email, Alamat
- Ketuk **Simpan** setelah mengisi

## 11. Assessment Mandiri (Level 1 & 2)

Hanya tersedia untuk murid Level 1 dan Level 2. Berisi penilaian diri tentang makharijul dan sifaatul huruf.

1. Buka Sidebar → **Assessment Mandiri**
2. Untuk setiap item, pilih:
   - **Paham** — sudah bisa melafazkan dengan benar
   - **Ragu** — belum yakin, perlu latihan lebih
   - **Belum** — perlu bimbingan guru
3. Pantau bar kemajuan di atas
4. Ketuk **Simpan Jawaban**

> 💡 Jawaban bisa diubah kapan saja sebelum sesi KBM. Guru bisa melihat hasilnya untuk mempersiapkan materi.

## 12. Install Aplikasi (PWA)

Portal bisa diinstall di HP seperti aplikasi biasa — tanpa perlu download dari App Store.

### iPhone (Safari)
1. Buka portal di Safari
2. Ketuk ikon **Bagikan** (kotak dengan panah ke atas) di toolbar bawah
3. Gulir ke bawah → ketuk **Tambahkan ke Layar Utama**
4. Ketuk **Tambahkan**

### Android (Chrome)
1. Buka portal di Chrome
2. Ketuk menu **⋮** (tiga titik) pojok kanan atas
3. Ketuk **Tambahkan ke layar utama** atau **Install app**
4. Konfirmasi → ikon portal muncul di home screen

**Keuntungan install:**
- Buka lebih cepat tanpa ketik URL
- Tampilan penuh tanpa address bar
- Tidak ada iklan
- Auto update — selalu versi terbaru

> 💡 Panduan ini juga tersedia di Sidebar Menu → **Install Aplikasi**.

## 14. Tips Umum

- **Mode Gelap** — ketuk ikon bulan/matahari di pojok kanan atas untuk beralih tema
