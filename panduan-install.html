Kenapa Harus Diinstall?

Perbedaan antara buka dari browser biasa vs install ke home screen sangat terasa.

⚡

Buka Langsung, Tanpa Ketik URL

Cukup ketuk ikon di home screen HP — langsung masuk portal. Tidak perlu buka browser, tidak perlu ingat alamat website.

📺

Tampilan Penuh Seperti Aplikasi Asli

Tidak ada address bar browser yang mengganggu. Seluruh layar untuk portal kamu — lebih nyaman dan profesional.

🔄

Tidak perlu update manual dari Play Store. Setiap kali dibuka, sudah versi terbaru — fitur baru langsung tersedia.

🔔

Hanya setelah install, kamu bisa aktifkan notifikasi. Pengingat dzikir pagi-petang, jadwal KBM, raport, & pengumuman — langsung ke HP.

📲 Panduan

Cara Install ke HP

Pilih sesuai HP dan browser yang kamu pakai.

⏱ ± 2 menit ✅ 5 langkah mudah 🆓 Tanpa biaya
💡
Gunakan Google Chrome Browser lain mungkin tidak punya opsi install. Jika belum punya Chrome, download dulu dari Play Store — gratis.
1
Buka portal di Chrome Android
Pastikan kamu sudah login ke akunmu. Tunggu halaman terbuka sepenuhnya.
2
Ketuk menu tiga titik ⋮
Di pojok kanan atas Chrome, ketuk tombol tiga titik vertikal (⋮).
3
Pilih "Tambahkan ke layar utama" atau "Install App"
Akan muncul pilihan Tambahkan ke layar utama atau Install App. Ketuk salah satunya. 💡 Jika muncul pop-up install otomatis di bawah layar, itu lebih mudah — langsung ketuk!
4
Ketuk "Install" atau "Tambahkan"
Konfirmasi — ikon portal akan muncul di home screen HP kamu.
5
Buka dari ikon baru dan aktifkan notifikasi
Ketuk ikon Rattililqur'an di home screen → tunggu beberapa detik → dialog notifikasi muncul → pilih "Ya, Aktifkan Notifikasi" → izinkan ketika HP bertanya.
💡
Wajib pakai Safari Install aplikasi di iPhone hanya bisa melalui browser Safari bawaan Apple. Chrome/Firefox di iPhone tidak bisa install.
1
Buka portal di Safari
Pastikan kamu membuka portal menggunakan aplikasi Safari (bukan Chrome atau browser lain).
2
Ketuk ikon Bagikan □↑
Di toolbar bawah Safari, ketuk ikon kotak dengan panah ke atas. Di iPhone terbaru, ini ada di bawah layar.
3
Gulir ke bawah → "Tambahkan ke Layar Utama"
Dari menu yang muncul, gulir ke bawah sampai menemukan Tambahkan ke Layar Utama → ketuk.
4
Ketuk "Tambahkan"
Konfirmasi nama aplikasi → ketuk Tambahkan di pojok kanan atas. Ikon portal muncul di home screen.
5
Buka dari ikon dan aktifkan notifikasi
Wajib buka dari ikon — bukan dari Safari! Ketuk ikon Rattililqur'an di home screen → login → tunggu dialog → aktifkan notifikasi. ⚠️ iPhone butuh iOS 16.4+ agar notifikasi berfungsi

Pertanyaan Umum

Hal yang sering ditanyakan.

Insya Allah, sangat aman. Portal ini berjalan di server terpercaya dengan enkripsi HTTPS. Data tersimpan di layanan database berstandar enterprise. Tidak ada data yang dijual atau dibagikan ke pihak luar.
Hampir tidak ada. Aplikasi ini hanya menyimpan beberapa file kecil untuk loading lebih cepat — biasanya kurang dari 5 MB, jauh lebih ringan dari aplikasi biasa yang bisa 50-100 MB.
Bisa kapan saja. Dari portal: Sidebar → Preferensi Notifikasi → matikan jenis yang tidak diinginkan. Atau dari pengaturan HP: Pengaturan → Notifikasi → Chrome/Safari → matikan.
Coba hard refresh dulu: tutup portal → buka lagi dari ikon. Jika masih tidak muncul, cek pengaturan HP: Pengaturan → Aplikasi → Chrome → Notifikasi → pastikan diizinkan. Jika masih bermasalah, hubungi admin.
Pastikan dua hal: (1) iPhone kamu iOS 16.4 atau lebih baru — cek di Pengaturan → Umum → Tentang → Versi iOS. (2) Buka portal dari ikon di home screen, bukan dari Safari browser. Notifikasi di iPhone hanya berfungsi dari ikon, bukan dari browser.
